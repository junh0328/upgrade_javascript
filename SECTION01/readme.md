## 🌟 변수

### Variable

값을 저장하는 영역

### 변수 사용 목적

변수에 저장된 값의 재사용

### 변수 선언 방법

```js
var book;
```

### 값을 변수에 할당하는 방법

```js
var book = '책';

>> '=' 오른쪽의 값(책)을 왼쪽 변수(book)에 할당
>> book 변수를 통해 '책'이라는 값을 재사용
```
